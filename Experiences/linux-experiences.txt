1. Linux Mint 更新有问题

终端中输入以下两条命令：
sudo rm /var/lib/apt/lists/* -vf
sudo apt-get update 


2. FreeBSD /etc/rc.conf

Manually config static IP address for a host

`-------------------------------------------
hostname="ysnBSD"
#ifconfig_em0="DHCP"
ifconfig_em0="172.18.190.38 netmask 255.255.255.0"
sshd_enable="YES"
mysql_enable="YES"
`-------------------------------------------
